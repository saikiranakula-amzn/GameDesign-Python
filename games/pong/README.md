# Pong

A classic Pong game implementation in Python using SimpleGUI.

## Description

Two-player paddle game where players control paddles to hit a ball back and forth. Features:
- Two-player gameplay
- Ball physics with increasing speed
- Score tracking
- Paddle collision detection

## How to Play

- **Player 1 (Left Paddle)**: W (up) and S (down)
- **Player 2 (Right Paddle)**: Up Arrow (up) and Down Arrow (down)
- **Goal**: Hit the ball past your opponent's paddle to score

## Running the Game

```bash
cd games/pong
python main.py
```

## Dependencies

- SimpleGUICS2Pygame (automatically handled in code)
- Python 3.6+
