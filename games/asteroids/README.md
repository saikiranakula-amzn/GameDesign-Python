# Asteroids (RiceRocks)

A classic Asteroids-style space shooter game implemented in Python using SimpleGUI.

## Description

Navigate your spaceship through an asteroid field, shooting asteroids while avoiding collisions. The game features:
- Spaceship movement with thrust and rotation
- Asteroid collision detection
- Score tracking
- Lives system
- Particle effects

## How to Play

- **Arrow Keys**: Rotate left/right and thrust forward/backward
- **Spacebar**: Fire missiles
- **Avoid**: Colliding with asteroids
- **Goal**: Destroy asteroids to increase your score

## Running the Game

```bash
cd games/asteroids
python main.py
```

## Dependencies

- SimpleGUICS2Pygame (automatically handled in code)
- Python 3.6+
