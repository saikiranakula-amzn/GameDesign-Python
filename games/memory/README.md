# Memory Game

A concentration-style memory game implemented in Python using SimpleGUI.

## Description

Test your memory by matching pairs of cards. Features:
- Grid of face-down cards
- Click to reveal cards
- Match pairs to keep them face-up
- Turn counter
- Win condition when all pairs are matched

## How to Play

- **Click**: Cards to reveal them
- **Goal**: Match all pairs in the fewest turns possible
- **Strategy**: Remember card positions to make matches

## Running the Game

```bash
cd games/memory
python main.py
```

## Dependencies

- SimpleGUICS2Pygame (automatically handled in code)
- Python 3.6+
