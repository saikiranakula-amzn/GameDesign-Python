# BlackJack

A digital implementation of the classic BlackJack card game in Python using SimpleGUI.

## Description

Play BlackJack against the dealer with standard rules. Features:
- Standard 52-card deck
- Dealer AI following standard BlackJack rules
- Hit/Stand gameplay
- Score tracking
- Card graphics

## How to Play

- **Goal**: Get as close to 21 as possible without going over
- **Hit**: Take another card
- **Stand**: Keep your current hand
- **Dealer**: Must hit on 16 or less, stand on 17 or more

## Running the Game

```bash
cd games/blackjack
python main.py
```

## Dependencies

- SimpleGUICS2Pygame (automatically handled in code)
- Python 3.6+
